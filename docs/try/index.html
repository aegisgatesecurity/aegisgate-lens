🛡️ Install the Lens first

This is a preview of the AegisGate Lens extension. To see the actual banner detection UI, install the Lens from the Chrome Web Store (one click), then come back to this page and try the test prompts below.

Why? The Lens is a browser extension — it can only run in the browser where it's installed. A web page can't load a Chrome extension (browsers prevent this for security reasons). The full experience requires the extension.
