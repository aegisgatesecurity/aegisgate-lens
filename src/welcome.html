AegisGate Lens

v0.0.0

AegisGate Lens protects you from accidentally pasting sensitive data (API keys, passwords, personal info) into AI chat tools like ChatGPT, Claude, and Gemini. It detects prompt injection attacks before they reach the AI. Detection runs entirely in your browser.

Privacy first. Default OFF. Your call.

AegisGate Lens works without sending any data anywhere. But if you opt in, your anonymized detection metadata helps us improve protection for everyone.

We NEVER send:
Your prompt text · URLs · Page content · User IDs · Session cookies

Choose your level of help

Detect-only (no telemetry)

AegisGate Lens protects you locally. No data leaves your browser. Recommended for maximum privacy.

Tier 1: Help improve detection (anonymized metadata)

Send anonymized detection metadata to AegisGate Security. We use this to identify new attack patterns and improve our detection. Recommended for most users.
✓ domain_hash (first 16 chars of SHA-256)
✓ category (pii_ssn, secret_jwt, prompt_injection, etc.)
✓ severity (low/medium/high/critical)
✓ user_action (send_anyway/edit/cancel/dismiss)
✓ timestamp + model version
✗ NO prompt text
✗ NO URLs
✗ NO page content
✗ NO personal identifiers

Tier 2: Full event details (power users)

Send richer event metadata for advanced threat intelligence. Still no prompt content or URLs. Includes detection timing, model consensus, and pattern IDs. Requires Tier 1.
✓ Everything in Tier 1, plus:
✓ detection timing (window index)
✓ model consensus (0-1)
✓ pattern fingerprint (NOT text)
✓ 30-day rolling count (anonymized)
✗ NO prompt text
✗ NO URLs
