// 5 PROMISES WE MAKE TO YOU
01

Your prompts never leave the browser.

Detection runs entirely client-side. No prompt text, no embeddings, no telemetry bytes are uploaded. The extension has host_permissions only for one endpoint that we never call with your data.

02

Detection in under 50 ms.

A 3-tier cascade (regex → TF-IDF → transformer) runs on every keystroke. Most prompts clear Tier 1 in <5 ms. Only ambiguous prompts reach the transformer, and even that path is local.

03

No account. No signup. No email.

Install and you're done. There is no onboarding flow, no license key, no "create your free account" wall. The source is Apache 2.0 — read every line on GitHub.

04

Open source. Auditable. Signed.

Every ML model bundle is Ed25519-signed with the public key shipped in the repo. You can verify what runs in your browser matches what we published — no silent model swaps, no remote-fetched code.

05

Honest failure modes.

When we're not sure, we say so. The banner shows the score, threshold, and per-model breakdown. False-positive dismissals are stored locally. CJK and Cyrillic are not covered yet — see the model card before relying on Lens for those scripts.
